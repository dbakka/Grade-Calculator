# Grade Calculator

## Description
This Grade Calculator is a simple web application that allows users to input a numerical mark and instantly see the corresponding letter grade along with personalized feedback. It's designed to provide quick and easy grade calculations for educators or students.

## Features
- Real-time grade calculation as the user types
- Color-coded grade display for quick visual feedback
- Personalized feedback messages based on the achieved grade
- Input validation to ensure marks are within the valid range (0-100)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## Setup and Installation
1. Clone this repository or download the ZIP file.
2. Extract the files if you downloaded a ZIP.
3. Ensure you have the following files in the same directory:
   - `index.html`
   - `styles.css`
   - `script.js`
4. Open `index.html` in a web browser to run the application.

## How to Use
1. Open the application in a web browser.
2. Enter a numerical mark between 0 and 100 in the input field.
3. The grade and feedback will update automatically as you type.
4. The grade will be displayed with a color corresponding to its level.
5. Read the personalized feedback below the grade.

## Grading Scale
- A+: 81-100
- A: 61-80
- B: 51-60
- C: 41-50
- D: 31-40
- E: 0-30

## Project Structure
- `index.html`: The main HTML file that structures the web page.
- `styles.css`: Contains all the CSS for styling the application.
- `script.js`: Includes the JavaScript code for calculating grades and updating the UI.

## Customization
You can easily customize the grading scale or feedback messages by modifying the `calculateGrade` function and the feedback conditions in the `updateResult` function within `script.js`.
